# Spring cloud function example
This is basic microservice example using Spring cloud function.

## Requisites
1. Download jdk8 and set it as your default.
2. Gradle 5.x or highter

## Tech

* Spring Cloud Function

### Run the application

```sh
$ ./gradlew shadowJar
```
